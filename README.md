# data-qa Skill

TPO 数据分析 Skill,支持对 xlsx/csv 文件进行自然语言问答。

## 功能特点

- **单月分析模式**: 默认返回最新月数据,支持指定月份查询
- **精确趋势对比**: 仅与相邻月对比、年度同比,不统计多月趋势
- **场景数据按实有**: 有啥说啥,不编造缺失场景
- **跨会话记忆**: 支持数据记忆和问答历史

## 业务领域知识

### 核心指标

| 指标 | 定义 | 计算公式 |
|------|------|---------|
| **TPO** | 人工进线量与完单量的比值 | `人工进线量 / 完单量 × 10000` |
| **YOY** | 年度同比(Year over Year) | `(本期值 - 上年同期值) / 上年同期值 × 100%` |
| **MoM** | 环比(Month over Month) | `(本期值 - 上月值) / 上月值 × 100%` |

### 场景名称映射

| 原始字段名 / 旧名称 | 当前业务名称 |
|---------------------|-------------|
| 微信渠道客诉 | **资产权益** |
| 二次号 | **售后服务** |

### 十大场景数据规则

数据文件中实际包含的场景会因月份而异:
- 2025/1-2025/10: 费用疑义、营销活动、二次号(3个场景)
- 2025/11-2026/3: 费用疑义、营销活动、售后服务、资产权益(4个场景)

**有啥说啥,不编造缺失场景**

## 使用方法

### 单月分析

```bash
python scripts/qa_engine_v3.py --file <csv_path> [--month <YYYY/M>]
```

**示例**:
```bash
# 分析最新月份
python scripts/qa_engine_v3.py --file D:\data\test.csv

# 分析指定月份
python scripts/qa_engine_v3.py --file D:\data\test.csv --month 2025/10
```

### 解析文件

```bash
python scripts/data_parser.py --file <xlsx_or_csv_path> --memory-dir memory/
```

### 记忆管理

```bash
# 查看所有数据记忆
python scripts/memory_manager.py --list-datasets

# 查看问答历史
python scripts/memory_manager.py --list-qa --limit 20

# 清除指定文件的记忆
python scripts/memory_manager.py --clear-dataset <file_hash>

# 搜索历史问答
python scripts/memory_manager.py --search-qa "<关键词>"
```

### 运行测试

```bash
python eval/run_evals.py
```

## 输出格式规范

**核心原则**:
1. **不累计**: 不计算多个月的累计变化
2. **精确描述**: 禁止使用模糊形容词
3. **默认最新月**: 用户未指定月份时,默认返回最新月份
4. **仅相邻对比**: 趋势仅与上个月和去年同期对比
5. **场景按实有**: 有啥说啥,不编造缺失场景

**输出结构**:
```
## 📊 查询结果

**问题**: [用户原始问题]
**月份**: [数据所属月份]

**结论**: [一句话核心答案]

**详细数据**:
[表格或列表]

**趋势对比**:
- 环比(vs上月): [数据]
- 同比(vs上年同月): [数据]
```

## 注意事项

1. 所有数据存储在本地,不上传云端,保护数据隐私
2. 渠道名称统一使用当前业务名称(资产权益、售后服务)
3. 2025/11 和 2025/12 的 YOY 为空属于正常,无需提示异常
4. 场景数据按实有场景展示,禁止编造其他场景
5. 数值精度默认保留2位小数
