# 输出格式规范

## 核心原则

### 1. 不累计（No Cumulative Calculations）

**禁止**：
- "累计增长 X%"
- "累计下降 Y%"
- "2025 年以来累计..."

**正确**：
- 直接展示当期数值
- 如果是趋势，明确对比月份："2026/3 vs 2026/2"

---

### 2. 精确描述（No Vague Adjectives）

**禁止的模糊词**：
- "剧烈" / "明显" / "显著" / "大幅" / "小幅"
- "持续" / "连续"（除非给出具体的连续月数，如"连续3个月"）
- "波动" / "起伏"（除非给出标准差或极差）

**正确写法**：
- "2026/3 TPO 为 13.31，较 2026/2（14.00）下降 4.9%"
- "售后服务在 2025/8-10 连续 3 个月 TPO 为 0"
- "资产权益 TPO 在 2025/5 达到最高值 2.59，2025/10 降至最低 0.12"

---

### 3. 默认最新月（Default to Latest Month）

用户未指定月份时，统一返回最新月份的数据（test.csv 为 2026/3）。

**示例**：
- 问题："各场景的 TPO 是多少？"
- 默认返回：2026/3 月的数据

如果用户指定月份，则返回指定月份的数据。

---

### 4. 仅相邻对比（Only Adjacent Comparisons）

趋势仅支持两种对比：
- **环比（MoM）**：当前月 vs 上个月
- **同比（YOY）**：当前月 vs 上年同月

**禁止**：
- "vs 2025/1"
- "vs 历史最高"
- "vs 任意历史月份"

**示例**：
- "2026/3 TPO 13.31，环比下降 4.9%（vs 2026/2 的 14.00），同比下降 18.0%（YOY）"
- 注意：YOY 直接读取数据列，无需重复计算

---

## 输出结构模板

```
## 📊 查询结果

**问题**：[用户原始问题]

**月份**：[数据所属月份，如"2026/3"（默认最新月）]

**结论**：[一句话核心答案，必须包含数值]

**详细数据**：
[表格或列表，展示所有场景的 TPO、YOY 值]

**趋势对比**：
- 环比（vs上月）：[当前月 TPO - 上月 TPO = 差值 / 上月 TPO × 100%]
- 同比（vs上年同月）：[读取 YOY 列，如"-18.0%"；无数据则写"-"]
```

---

## 结论部分的精确格式

**错误示例**：
- "平台总 TPO 持续下降，趋势明显"
- "资产权益波动剧烈"

**正确示例**：
- "2026/3 平台总 TPO 为 13.31，环比下降 4.9%，同比下降 18.0%"
- "资产权益在 2026/3 的 TPO 为 0.96，较 2025/10（0.12）增长 700%"

---

## 数值精度

- TPO 保留 2 位小数
- 百分比保留 1 位小数（如 -18.0%）
- 如果原始数据为整数，保持整数格式（如完单量 28,616,498）

---

## 空值处理

- YOY 为空时，显示为 "-"
- TPO 为空或无法计算时，显示为 "-"
- 不对空值做异常提示，除非用户明确问异常检测
