# 查询意图映射表（更新版）

## 意图分类

### 1. 单点查询（默认场景）

**触发词**：
- "各场景的 TPO"
- "最新月数据"
- "X 月的 TPO"
- "所有场景的 YOY"

**处理**：
- 返回指定月份的数据（默认最新月）
- 不做任何累计或聚合

---

### 2. 对比查询

**触发词**：
- "X 场景和 Y 场景的对比"
- "哪个场景 TPO 更高"
- "X 场景 vs Y 场景"

**处理**：
- 返回指定月份两个场景的 TPO 值
- 可补充绝对差异

---

### 3. 环比查询（MoM）

**触发词**：
- "X 场景的环比"
- "vs 上月"
- "环比变化"

**处理**：
- 计算当前月 vs 上个月的差异
- 公式：`(当前月 - 上月) / 上月 × 100%`

---

### 4. 同比查询（YOY）

**触发词**：
- "X 场景的同比"
- "vs 上年同月"
- "年度环比"

**处理**：
- 直接读取 YOY 列
- 无数据时显示 "-"

---

### 5. 排名查询

**触发词**：
- "TPO 最高的场景"
- "TPO 最低的场景"
- "TOP N"

**处理**：
- 对指定月份的场景排序
- 返回前 N 个

---

### 6. 异常检测

**触发词**：
- "TPO=0 的场景"
- "YOY < -30% 的场景"
- "异常数据"

**处理**：
- 按指定月份筛选
- 不使用模糊词（如"异常"），直接输出筛选结果

---

## 已删除的意图类型

以下意图不再支持（违反"不累计"、"不模糊描述"原则）：

- ~~聚合查询~~（如"各场景总进线量"）
- ~~趋势查询~~（如"月度变化趋势"、"累计趋势"）
- ~~波动分析~~（如"波动剧烈"、"趋势明显"）

---

## 关键词映射

| 用户输入 | 映射到意图 | 参数提取 |
|---------|-----------|---------|
| "最新月" | `latest_month` | month=最新 |
| "X 月" | `single_month` | month=X |
| "环比" | `mom` | - |
| "同比" | `YOY` | - |
| "最高" | `max` | sort=desc |
| "最低" | `min` | sort=asc |
| "TOP N" | `top_n` | n=N |
| "对比" | `compare` | scenes=[X, Y] |
